# Repacker

Automated tool to convert Recovery ROMs into Fastboot flashable ZIPs. Includes a web interface to trigger builds through GitHub Actions.

## Features
- Converts payload-based recovery ROMs to Fastboot format.
- Supports Direct and Google Drive download links.
- Automated uploads to Gofile.io.
- Automatic GitHub Release creation with download links.
- Configurable device names and compatibility lists.
